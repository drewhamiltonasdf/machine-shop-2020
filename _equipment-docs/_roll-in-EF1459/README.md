### Specs:

  - 1HP Motor, 3 Phase
  - Blade Speed Adjustable
  - Replacement blade info: (10 ft. blade length)

&nbsp;
&nbsp;
  
### Operation

  - Make sure Tension Handle is Tight
  - Select correct speed for material (see chart on machine)
  - Make sure workpiece is securely clamped
